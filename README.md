# GPX file analyzer

## Installation

1. Clone the repository
    ```sh
    git clone git@github.com:ValentinPeltier/gpx-analyzer.git
    ```
1. Install dependencies
    ```sh
    npm install
    ```

## Usage

- For development:
    1. Start the development server
        ```sh
        npm run dev
        ```
    1. Navigate to http://localhost:3000

- For production:
    1. Build the server
        ```sh
        npm run build
        ```
    1. Start the server
        ```sh
        npm run start
        ```
    1. Navigate to http://localhost:3000
